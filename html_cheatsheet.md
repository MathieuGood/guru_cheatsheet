# HTML Cheat Sheet

```html
<!DOCTYPE html>
<html lang="fr">

    <head>
        <meta charset="utf-8">
        <title>My page title</title>
        <link href="style.css" rel="stylesheet">
    </head>

    <body>

        <h1>Lists</h1>
            <h2>Unordered list</h2>
                <ul>
                    <li>First item</li>
                    <li>Second item</li>
                </ul>
            <h2>Ordered list</h2>
                <ol>First item</ol>
                <ol>Second item</ol>

        <h1 id ="my_anchor">Links and anchors</h1>
            <a href="#my_anchor">Go to anchor</a>
            <a href="page.html#my_anchor">Reach anchor on different page</a>
            <a href="page.html" target="_blank">Open in new tab</a>




    </body>

</html>


```
